 Titanic Survival Prediction 🚢

This project predicts whether a passenger survived using Machine Learning.

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting

Run Project

bash
pip install -r requirements.txt
python titanic_prediction.py
